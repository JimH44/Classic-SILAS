WordSend_Styles.txt -- control file for converting WordSend style names
to those in USFM.sty, version 2.208 of Apr 29, 2009.

Comment lines have no semi-colons in them.
Change lines have the old stylename that should be replaced first,
then after the semicolon the new replacement name.

     WordSend has ps produce the same as p, namely "Normal"
     so there is no way to get paragraphs in the style

          "OBSOLETE ps - Paragraph - No Break with Next Paragraph"
